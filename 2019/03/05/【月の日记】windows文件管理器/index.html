【月の日记】windows文件管理器
发表于 2019-03-05 | 更新于 2019-03-05
月の日记
日记 | 生活 | 电脑

我是一个爱折腾的人，特别喜欢去研究如何优化自己心爱的电脑，怎么才能让它的性能发挥到极致，怎么让自己用得舒服（主要还是怎么去装13。。。）

昨天下了个XYplorer玩了一下，加上之前我一直在使用的Q-Dir与Total Commander，已经是第三个window系统的文件管理器了。虽然window系统自带的文件管理这么多年来说，一点长进都没有，不得不逼我去找更好用的文件管理器。虽然前面两个都已经完全够用了，但每次在网上瞎逛的时候，看到有介绍软件的文章都细细研究一番，看看有没有好用的软件。

结果昨天又心动用上了XYplorer，结果发现还挺不错。但三个文件管理器实在有点太多了，如果三个一起用的话，不但效果没有挺高，反而还会影响自己日常的操作。因为三个软件的操作都有些异常（尤其是Total Commander，一开始总觉得它是不是反人类的设计）。所以还是狠下心来放弃一个吧。

但是放弃哪一个呢？

  • Q-Dir是我最初开始使用的，具体哪一年开始用就忘记了，但至少已经也有7、8年了吧。说真的，放弃它有点不拾得，但界面有点丑是真的，而且就是分成不多个视图框，复制粘贴就很方便与体积小的优点，但其他没有什么特别。
  • Total Commander的话是用来装13用的，感觉键盘党的最爱吧，当然可以安装很多插件，完全体的它，功能强大到吖~~
  • XYplorer是刚刚开始使用，具体还没有探索透，但从界面来说，比Q-Dir好睇多了，还有一些如批量改名、文件预览、搜索等功能，也有双窗格功能，需要没有Q-Dir可以分成很多个窗口，但用多了之后慢慢发现，其实日常操作中，双窗口已经满足复制粘贴的操作需求了

所以，最后只能暂时跟Q-Dir讲再见了，但我还是会继续留意它官网的更新，哪一天如果功能得到大幅度提升，我还是会再使用的！
