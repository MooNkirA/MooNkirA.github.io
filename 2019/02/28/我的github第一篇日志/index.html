个人的在github page搭建的blog第一篇日记

个人简介

本人是一个有梦想但总是有拖延症的人，一直都想搞个个人blog，但又没有技术。就去一些blog网站上写文章，但眼看blog网站越来越少人用，网站也是倒了一个接一个。

其实个人也是一个爱折腾的人，又有一些强迫症，一直在寻找好的blog网站，可以发布一些个人的生活、平时的感悟、自己学习的技术（实在没有什么技术，都是ctrl+c\ctrl+v而已）。现在终于自己搞个github的个人网页。虽然页面不是很好看（也是抄的），但总感觉比以前写blog更有兴趣了，毕竟是自己去搭建的。往后的日子里，估计也会继续去折腾，慢慢将页面一点点优化吧

总结

希望这是我的新的开始，而不是又一次的结束！
